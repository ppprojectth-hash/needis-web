## 1. ภาพรวมเว็บไซต์

เว็บไซต์ **Neediss** เป็นเว็บไซต์โปรโมทสินค้าด้านเครื่องมือวัด เครื่องมืออุตสาหกรรม เครื่องมือวิทยาศาสตร์ อุปกรณ์ห้องปฏิบัติการ และเครื่องมือตรวจวัดสิ่งแวดล้อม

### ฟีเจอร์หลักของเว็บไซต์

| ฟีเจอร์ | คำอธิบาย |
|---|---|
| เรียกดูสินค้า | ดูสินค้าทั้งหมด กรองตามหมวดหมู่ ค้นหาตามชื่อ/แบรนด์ |
| รายละเอียดสินค้า | ดูรูปภาพ สเปค โบรชัวร์ และข้อมูลครบถ้วน |
| ใบเสนอราคา | เพิ่มสินค้าและบริการลงตะกร้า แล้วส่งคำขอใบเสนอราคา |
| บริการ | ดูบริการที่บริษัทให้บริการ เช่น สอบเทียบ ซ่อมบำรุง |
| กิจกรรม | ข่าวสารและกิจกรรมล่าสุดของบริษัท |
| ติดต่อ | แบบฟอร์มติดต่อบริษัท |
| รองรับ 2 ภาษา | ภาษาไทยและภาษาอังกฤษ |

## 3. หน้าแรก (Home Page)

เส้นทาง: `/` หรือ `/Home`

หน้าแรกประกอบด้วยส่วนต่างๆ ดังนี้:

### 3.1 แถบนำทาง (Navigation Bar)

อยู่ด้านบนสุดของทุกหน้า ประกอบด้วย:

- **โลโก้บริษัท** — คลิกเพื่อกลับหน้าแรก
- **เมนูหลัก:**
  - Home — หน้าแรก
  - Products — สินค้าทั้งหมด
  - Services — บริการ
  - Activity — กิจกรรม/ข่าวสาร
  - About — เกี่ยวกับบริษัท
  - Contact — ติดต่อเรา
- **ปุ่ม Request Quote** — ขอใบเสนอราคา
- **ปุ่มเปลี่ยนภาษา** TH / EN

### 3.2 Hero Banner (แบนเนอร์หลัก)

แบนเนอร์ขนาดใหญ่ด้านบนสุด แสดงภาพหรือวิดีโอ พร้อมข้อความโปรโมท มีปุ่มลิงก์ไปยังสินค้าหรือบริการ หากมีหลายแบนเนอร์ จะสไลด์อัตโนมัติ

### 3.3 หมวดหมู่สินค้า (Product Categories)

แสดงหมวดหมู่สินค้าทั้งหมด คลิกที่หมวดหมู่เพื่อกรองสินค้าในหมวดนั้น

### 3.4 สินค้าแนะนำ (Featured Products)

แสดงสินค้าที่น่าสนใจ จัดกลุ่มตามหมวดหมู่

### 3.5 บริการ (Services)

แสดงบริการหลักของบริษัท เช่น การจำหน่าย สอบเทียบ ซ่อมบำรุง

### 3.6 กิจกรรม/ข่าวสาร (Activity)

แสดงข่าวสารและกิจกรรมล่าสุด (หากมีข้อมูล)

### 3.7 Call-to-Action (CTA)

ส่วนด้านล่างที่เชิญชวนให้ติดต่อหรือขอใบเสนอราคา

### 3.8 Footer

แสดงข้อมูลบริษัท ที่อยู่ เบอร์โทร อีเมล และ Social Media

## 4. การเรียกดูสินค้า (Products)

เส้นทาง: `/Product`

### ขั้นตอน

1. คลิกเมนู **Products** บนแถบนำทาง
2. ระบบจะแสดงสินค้าทั้งหมดในรูปแบบ Grid
3. แต่ละการ์ดสินค้าจะแสดง:
   - รูปภาพสินค้า
   - ชื่อสินค้า
   - แบรนด์/รุ่น
   - คำอธิบายสั้น
   - ราคา (ถ้าบริษัทกำหนดให้แสดง)
   - ปุ่ม **View Details** / **ดูรายละเอียด**

### การเรียกดูตามหมวดหมู่

1. บนแถบด้านซ้าย (หรือด้านบน บนมือถือ) จะมีรายการหมวดหมู่
2. คลิกชื่อหมวดหมู่เพื่อกรองสินค้า
3. คลิก **All / ทั้งหมด** เพื่อดูสินค้าทั้งหมด

---

## 5. การค้นหาและกรองสินค้า

### การค้นหาด้วยคำค้น

1. บนหน้าสินค้า จะมีช่อง **ค้นหา (Search)**
2. พิมพ์ชื่อสินค้า แบรนด์ หรือรุ่น
3. กด **Enter** หรือคลิกปุ่มค้นหา
4. ระบบจะแสดงเฉพาะสินค้าที่ตรงกับคำค้น

### การกรองตามหมวดหมู่

- เส้นทาง: `/Product?category=ชื่อ-slug`
- ตัวอย่าง: `/Product?category=measuring-instruments`

คลิกหมวดหมู่ที่ต้องการ ระบบจะกรองสินค้าให้อัตโนมัติ

### การเรียงลำดับ

หากมีตัวเลือกการเรียงลำดับ สามารถเรียงตาม:
- ลำดับที่กำหนด (Default)
- ชื่อ A-Z
- ราคา (ถ้าแสดง)

## 6. การดูรายละเอียดสินค้า

เส้นทาง: `/Product/Detail/{slug}`

### ขั้นตอน

1. คลิกสินค้าที่สนใจจากหน้ารายการสินค้า
2. ระบบจะเปิดหน้ารายละเอียดสินค้า ซึ่งประกอบด้วย:

### ข้อมูลที่แสดงในหน้ารายละเอียด

| ส่วน | ข้อมูล |
|---|---|
| รูปภาพ | รูปภาพหลัก และรูปภาพเพิ่มเติม (คลิกเปลี่ยนรูปได้) |
| ชื่อสินค้า | ชื่อภาษาไทย/อังกฤษ |
| แบรนด์ / รุ่น | ชื่อแบรนด์และรุ่นของสินค้า |
| SKU | รหัสสินค้า |
| ราคา | แสดงถ้าบริษัทกำหนดให้แสดง |
| คำอธิบาย | คำอธิบายสั้นและคำอธิบายเต็ม |
| สเปค | ตารางสเปคทางเทคนิค |
| โบรชัวร์ | ปุ่มดาวน์โหลด PDF (ถ้ามี) |
| ปุ่มดำเนินการ | ขอใบเสนอราคา / ติดต่อเรา |

### การดูรูปภาพเพิ่มเติม

- คลิกที่รูปภาพเล็กด้านล่างเพื่อเปลี่ยนรูปหลัก

### การดาวน์โหลดโบรชัวร์

1. เลื่อนลงมาที่ส่วนโบรชัวร์
2. คลิกปุ่ม **Download Brochure / ดาวน์โหลดโบรชัวร์**
3. ไฟล์ PDF จะถูกดาวน์โหลด

## 7. การเพิ่มสินค้าลงตะกร้าใบเสนอราคา

### ขั้นตอน

1. เปิดหน้ารายละเอียดสินค้า
2. คลิกปุ่ม **Add to Quote / เพิ่มในใบเสนอราคา**
3. ระบบจะแสดงข้อความยืนยันว่าเพิ่มสำเร็จ
4. ไอคอนตะกร้าบนแถบนำทางจะแสดงจำนวนสินค้าที่เพิ่ม

### การดูตะกร้าใบเสนอราคา

เส้นทาง: `/QuotationCart`

1. คลิกไอคอนตะกร้าบนแถบนำทาง หรือเมนู Quote Cart
2. ระบบจะแสดงรายการสินค้าที่เพิ่มทั้งหมด

### การจัดการในตะกร้า

| การดำเนินการ | วิธี |
|---|---|
| เปลี่ยนจำนวน | แก้ไขตัวเลขในช่องจำนวน |
| ลบสินค้า | คลิกปุ่ม Remove / ลบ |
| ล้างตะกร้า | คลิก Clear All / ล้างทั้งหมด |
| ส่งคำขอ | คลิก Request Quotation / ส่งคำขอใบเสนอราคา |

## 8. การเรียกดูบริการ (Services)

เส้นทาง: `/Services`

## 9. การเพิ่มบริการลงตะกร้าใบเสนอราคา

### ขั้นตอน

1. เปิดหน้ารายละเอียดบริการ
2. คลิกปุ่ม **Add to Quote / เพิ่มในใบเสนอราคา**
3. บริการจะถูกเพิ่มเข้าตะกร้าร่วมกับสินค้า
4. ตรวจสอบตะกร้าได้ที่ `/QuotationCart`

## 10. การส่งคำขอใบเสนอราคา (Quotation)

เส้นทาง: `/Quotation/Create`

### ขั้นตอน

**วิธีที่ 1: ผ่านตะกร้าใบเสนอราคา**

1. เพิ่มสินค้า/บริการลงตะกร้าก่อน
2. คลิกไอคอนตะกร้า หรือไปที่ `/QuotationCart`
3. ตรวจสอบรายการ
4. คลิก **Request Quotation / ส่งคำขอ**
5. กรอกฟอร์ม:
   - ชื่อ-นามสกุล *
   - อีเมล *
   - เบอร์โทรศัพท์ *
   - ชื่อบริษัท/องค์กร
   - ข้อความเพิ่มเติม
6. คลิก **Submit / ส่ง**

**วิธีที่ 2: ส่งโดยตรง**

1. คลิกปุ่ม **Request Quote** บนแถบนำทาง หรือไปที่ `/Quotation/Create`
2. กรอกข้อมูลส่วนตัวและรายละเอียดสินค้าที่ต้องการ
3. คลิก **Submit / ส่ง**

### หลังจากส่งคำขอ

- ระบบจะแสดงข้อความยืนยัน
- ทีมงานจะติดต่อกลับภายใน 24 ชั่วโมง
- ท่านอาจได้รับอีเมลยืนยัน (ถ้าระบบส่งอีเมล)

## 11. การติดต่อบริษัท (Contact)

เส้นทาง: `/Contact`

### ขั้นตอน

1. คลิกเมนู **Contact** บนแถบนำทาง

   - ที่อยู่

   - แผนที่ (ถ้ามี)

3. กรอกแบบฟอร์มติดต่อ:
   - ชื่อ *
   - อีเมล *
   - เบอร์โทร
   - หัวข้อ
   - ข้อความ *
4. คลิก **Send / ส่งข้อความ**

> ทีมงานจะตอบกลับโดยเร็วที่สุด

## 12. การเปลี่ยนภาษา (Language Switcher)

เว็บไซต์รองรับ 2 ภาษา: **ภาษาไทย (TH)** และ **ภาษาอังกฤษ (EN)**

### ขั้นตอน

1. มองหาปุ่มเปลี่ยนภาษาที่มุมขวาบนของแถบนำทาง
2. คลิก **TH** เพื่อเปลี่ยนเป็นภาษาไทย
3. คลิก **EN** เพื่อเปลี่ยนเป็นภาษาอังกฤษ
4. หน้าจะรีโหลดและแสดงเนื้อหาในภาษาที่เลือก

> **หมายเหตุ:** ภาษาที่เลือกจะถูกจำไว้ในระหว่างการเยี่ยมชม

## 13. คำถามที่พบบ่อย (FAQ)

**Q: ทำไมสินค้าบางรายการไม่แสดงราคา?**  
A: บริษัทอาจกำหนดให้ราคาบางรายการต้องติดต่อสอบถาม กรุณากรอกฟอร์มขอใบเสนอราคาหรือติดต่อโดยตรง

**Q: สามารถเพิ่มสินค้าได้กี่รายการในตะกร้า?**  
A: ไม่มีจำกัดจำนวนรายการ แต่แนะนำให้ไม่เกิน 50 รายการต่อครั้ง

**Q: หลังจากส่งคำขอใบเสนอราคาแล้วจะได้รับการตอบกลับอย่างไร?**  
A: ทีมงานจะติดต่อกลับทางอีเมลหรือโทรศัพท์ภายใน 24 ชั่วโมง (วันทำการ)

**Q: สามารถดาวน์โหลดโบรชัวร์สินค้าได้หรือไม่?**  
A: ได้ สินค้าที่มีโบรชัวร์จะมีปุ่ม Download Brochure ในหน้ารายละเอียดสินค้า

**Q: เว็บไซต์รองรับมือถือหรือไม่?**  
A: รองรับทุกขนาดหน้าจอ ทั้ง Desktop, Tablet และ Smartphone

**Q: ไม่สามารถส่งแบบฟอร์มได้ ต้องทำอย่างไร?**  
A: ตรวจสอบว่ากรอกข้อมูลที่มีเครื่องหมาย * (จำเป็น) ครบทุกช่อง และอีเมลต้องเป็นรูปแบบที่ถูกต้อง เช่น name@example.com

**Q: ข้อมูลที่ส่งไปปลอดภัยหรือไม่?**  
A: ข้อมูลทั้งหมดถูกส่งผ่าน HTTPS (เข้ารหัส) และเก็บไว้ในระบบของบริษัทอย่างปลอดภัย

**Q: ไม่พบสินค้าที่ต้องการ ต้องทำอย่างไร?**  
A: กรุณาติดต่อทีมงานผ่านหน้า Contact หรือโทรตามเบอร์ที่แสดงในเว็บไซต์
