## 1. ภาพรวมเว็บไซต์

**Neediss** เป็นเว็บไซต์โปรโมทสินค้าด้านเครื่องมือวัด เครื่องมืออุตสาหกรรม เครื่องมือวิทยาศาสตร์ อุปกรณ์ห้องปฏิบัติการ และเครื่องมือตรวจวัดสิ่งแวดล้อม

| ฟีเจอร์ | คำอธิบาย |
|---|---|
| เรียกดูสินค้า | ดูสินค้าทั้งหมด กรองตามหมวดหมู่ ค้นหาตามชื่อ/แบรนด์ |
| รายละเอียดสินค้า | ดูรูปภาพ ตารางสเปค โบรชัวร์ |
| ใบเสนอราคา | เพิ่มสินค้า/บริการลงตะกร้า ส่งคำขอ |
| บริการ | สอบเทียบ ซ่อมบำรุง ให้คำปรึกษา |
| กิจกรรม | ข่าวสารและกิจกรรมล่าสุด |
| ติดต่อ | แบบฟอร์มติดต่อบริษัท |
| 2 ภาษา | ภาษาไทยและภาษาอังกฤษ |

## 4. การใช้งานหน้าแรก (Home)

เส้นทาง: `/` หรือ `/Home`

หน้าแรกประกอบด้วยส่วนดังนี้:

### แถบนำทาง (Navigation Bar)

- **โลโก้** — คลิกเพื่อกลับหน้าแรก
- **เมนูหลัก:** Home | Products | Services | Activity | About | Contact
- **Request Quote** — ขอใบเสนอราคาโดยตรง
- **TH / EN** — เปลี่ยนภาษา

### Hero Banner

แบนเนอร์ขนาดใหญ่ด้านบน แสดงภาพหรือวิดีโอ หากมีหลายแบนเนอร์จะสไลด์อัตโนมัติ

### หมวดหมู่สินค้า

แสดงหมวดหมู่สินค้าทั้งหมด — คลิกเพื่อกรองสินค้าในหมวดนั้น

### สินค้าแนะนำ (Featured Products)

สินค้าที่น่าสนใจ จัดกลุ่มตามหมวดหมู่

### บริการ (Services)

บริการหลักของบริษัท

### Footer

ที่อยู่ เบอร์โทร อีเมล และลิงก์ Social Media

---

## 5. การเรียกดูสินค้า (Products)

เส้นทาง: `/Product`

### ขั้นตอน

1. คลิกเมนู **Products** บนแถบนำทาง
2. ระบบแสดงสินค้าทั้งหมดในรูปแบบ Grid
3. แต่ละการ์ดสินค้าแสดง:
   - รูปภาพ
   - ชื่อสินค้า / แบรนด์ / รุ่น
   - คำอธิบายสั้น
   - ราคา (ถ้ากำหนดให้แสดง)
   - ปุ่ม **View Details / ดูรายละเอียด**

### เรียกดูตามหมวดหมู่

1. คลิกหมวดหมู่จากแถบด้านซ้าย (หรือด้านบนบนมือถือ)
2. ระบบกรองสินค้าในหมวดนั้นทันที
3. คลิก **All / ทั้งหมด** เพื่อดูสินค้าทั้งหมด

---

## 6. การค้นหาและกรองสินค้า

### ค้นหาด้วยคำค้น

1. พิมพ์ชื่อสินค้า แบรนด์ หรือรุ่น ในช่อง **Search**
2. กด **Enter** หรือคลิกปุ่มค้นหา
3. ระบบแสดงเฉพาะสินค้าที่ตรง

### กรองตามหมวดหมู่

เส้นทางตัวอย่าง: `/Product?category=measuring-instruments`

คลิกหมวดหมู่ที่ต้องการ ระบบกรองอัตโนมัติ

---

## 7. หน้ารายละเอียดสินค้า

เส้นทาง: `/Product/Detail/{slug}`

### ขั้นตอน

1. คลิกสินค้าที่สนใจจากหน้ารายการ
2. ระบบเปิดหน้ารายละเอียด

### ข้อมูลที่แสดง

| ส่วน | ข้อมูล |
|---|---|
| รูปภาพ | รูปหลัก + รูปเพิ่มเติม (คลิกเปลี่ยนได้) |
| ชื่อ / แบรนด์ / รุ่น | ชื่อสินค้าทั้ง 2 ภาษา |
| SKU | รหัสสินค้า |
| ราคา | แสดงถ้ากำหนด |
| คำอธิบาย | สั้นและเต็ม |
| สเปค | ตารางสเปคทางเทคนิค |
| โบรชัวร์ | ปุ่มดาวน์โหลด PDF (ถ้ามี) |

### ดาวน์โหลดโบรชัวร์

คลิกปุ่ม **Download Brochure / ดาวน์โหลดโบรชัวร์** — ไฟล์ PDF จะถูกดาวน์โหลด

## 8. ตารางสเปคสินค้า

ตารางสเปคแสดงข้อมูลทางเทคนิคในรูปแบบโครงสร้าง:

- **จัดกลุ่มตามหัวข้อ** เช่น ทั่วไป, ไฟฟ้า, มิติ
- **ไฮไลท์สเปคสำคัญ** — แสดงเด่นกว่ารายการอื่น
- **หน่วยวัด** แสดงถัดจากค่า (เช่น V, A, mm, kg)
- **รองรับ 2 ภาษา** — แสดงตามภาษาที่เลือก

## 9. การเพิ่มสินค้าลงตะกร้าใบเสนอราคา

### ขั้นตอน

1. เปิดหน้ารายละเอียดสินค้า
2. คลิกปุ่ม **Add to Quote / เพิ่มในใบเสนอราคา**
3. ระบบแสดงข้อความยืนยัน
4. ไอคอนตะกร้าบน Navbar อัปเดตจำนวน

### จัดการตะกร้า

เส้นทาง: `/QuotationCart`

| การดำเนินการ | วิธี |
|---|---|
| เปลี่ยนจำนวน | แก้ไขตัวเลขในช่องจำนวน |
| ลบสินค้า | คลิกปุ่ม Remove |
| ล้างตะกร้า | คลิก Clear All |
| ส่งคำขอ | คลิก Request Quotation |

---

## 10. การเรียกดูบริการ (Services)

เส้นทาง: `/Services`

1. คลิกเมนู **Services** บน Navbar
2. ระบบแสดงบริการทั้งหมด เช่น:
   - การจำหน่ายเครื่องมือ
   - บริการสอบเทียบ (Calibration)
   - บริการซ่อมบำรุง (Maintenance & Repair)
   - ให้คำปรึกษาทางเทคนิค
3. คลิกบริการเพื่อดูรายละเอียด

---

## 11. การเพิ่มบริการลงตะกร้าใบเสนอราคา

1. เปิดหน้ารายละเอียดบริการ
2. คลิกปุ่ม **Add to Quote / เพิ่มในใบเสนอราคา**
3. ตรวจสอบตะกร้าที่ `/QuotationCart`

> ตะกร้าสามารถมีทั้งสินค้าและบริการพร้อมกัน

---

## 12. การส่งคำขอใบเสนอราคา

เส้นทาง: `/Quotation/Create`

### วิธีที่ 1: ผ่านตะกร้า

1. เพิ่มสินค้า/บริการลงตะกร้า
2. ไปที่ `/QuotationCart`
3. ตรวจสอบรายการ
4. คลิก **Request Quotation**
5. กรอกฟอร์ม:
   - ชื่อ-นามสกุล *
   - อีเมล *
   - เบอร์โทรศัพท์ *
   - ชื่อบริษัท/องค์กร
   - ข้อความเพิ่มเติม
6. คลิก **Submit / ส่ง**

### วิธีที่ 2: ส่งโดยตรง

ไปที่ `/Quotation/Create` หรือคลิกปุ่ม **Request Quote** บน Navbar

### หลังจากส่งคำขอ

- ระบบแสดงข้อความยืนยัน
- ทีมงานติดต่อกลับภายใน 24 ชั่วโมง (วันทำการ)

   - อีเมล *
   - เบอร์โทร

   - ข้อความ *
4. คลิก **Send / ส่งข้อความ**

## 15. คำถามที่พบบ่อย (FAQ)

**Q: ทำไมสินค้าบางรายการไม่แสดงราคา?**
A: ราคาบางรายการต้องติดต่อสอบถาม กรุณาขอใบเสนอราคาหรือติดต่อโดยตรง

**Q: เพิ่มสินค้าได้กี่รายการในตะกร้า?**
A: ไม่จำกัด แนะนำไม่เกิน 50 รายการต่อครั้ง

**Q: หลังส่งคำขอใบเสนอราคาจะได้รับการตอบกลับเมื่อไหร่?**
A: ภายใน 24 ชั่วโมง (วันทำการ) ทางอีเมลหรือโทรศัพท์

**Q: ดาวน์โหลดโบรชัวร์ได้หรือไม่?**
A: ได้ สินค้าที่มีโบรชัวร์จะมีปุ่ม Download Brochure ในหน้ารายละเอียด

**Q: รองรับมือถือหรือไม่?**
A: รองรับทุกขนาดหน้าจอ ทั้ง Desktop, Tablet และ Smartphone

**Q: ส่งฟอร์มไม่ได้ ต้องทำอย่างไร?**
A: ตรวจสอบว่ากรอกฟิลด์ที่มีเครื่องหมาย * ครบ และอีเมลถูกต้อง เช่น name@example.com

**Q: ไม่พบสินค้าที่ต้องการ ทำอย่างไร?**
A: ติดต่อทีมงานผ่านหน้า Contact หรือโทรตามเบอร์ที่แสดงใน Footer

```
Browser / Device: [เช่น Chrome บน Windows 11]

ขั้นตอนที่ทำ:
1.
2.
3.

ผลที่คาดหวัง: [สิ่งที่ควรเกิดขึ้น]
ผลที่เกิดขึ้นจริง: [สิ่งที่เกิดขึ้น]

ความรุนแรง: [ ] Critical  [ ] High  [ ] Medium  [ ] Low

Screenshot/วิดีโอ: [แนบถ้ามี]
```

- ส่งให้ทีมพัฒนาโดยตรง
